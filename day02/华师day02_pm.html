一、JavaScript:
  • JavaScript 是脚本语言
    • JavaScript 是一种轻量级的编程语言。
    • JavaScript 是可插入 HTML 页面的编程代码。
    • JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。
    • JavaScript 很容易学习。
  • <script> 标签
    • 如需在 HTML 页面中插入 JavaScript，请使用 <script> 标签。
    • <script> 和 </script> 会告诉 JavaScript 在何处开始和结束。
    • <script> 和 </script> 之间的代码行包含了 JavaScript：
      如：<script> alert("My First JavaScript"); </script>
  • 使用方式：
    • 1）在标签内直接写js代码：
      •     <input type="button" onclick="alert('Hello!')" value="sayHello">
      •     <!-- onclick 单机事件  alert(‘弹出的内容’) 弹出  -->
    • 2）在标签内调用函数

function say() {
         alert("HelloWrold.");
    }/*该函数在<script>标签内定义*/
<input type="button" onclick="say()" value="sayHello2">

                    3）在js中给控件添加事件

window.onload = function() { /*window.onload() 是在页面元素框架加载完之后运行  */
         /* 找到 btn03  并加单击事件 */
         var btn03 = document.getElementById("btn03"); /* 从文档对象中根据id查询一个元素 */
         btn03.onclick = function() { /*给单击事件赋值  */
             alert("hello World.");
         }
    }/*该函数在<script>标签内定义*/


<input type="button" id="btn03" value="sayHello3">
    
    4）引用外部的js文件：

<script type="text/javascript" src="jquery-1.7.2.js"></script><!--引用外部的js文件 -->

2、文档对象模型
  • 文档对象模型Document Object Model
    • DOM定义了访问和处理 HTML 文档的标准方法。是W3C国际组织制定的统一标准，在很多计算机语言中都有不同实现
  • 根据 W3C 的 HTML DOM 标准，HTML 文档中的所有内容都是节点：
    • 整个文档是一个文档节点
    • 每个 HTML 元素是元素节点
    • HTML 元素内的文本是文本节点
    • 每个 HTML 属性是属性节点
    • 注释是注释节点
  • 几个方法：
    • 在整个文档范围内查询元素节点
    • 根据id值查询【返回一个具体节点】:  
    •       document.getElementById(“id值”)
    • 根据标签名查询【返回节点数组】:
    •       document.getElementsByTagName(“标签名”)
    • 根据name属性值查询【返回节点数组】:
    •       document.getElementsByName(“name值”)
    • 在具体元素节点范围内查找子节点
      • 查找全部子节点【返回节点数组】:
    •       element.childNodes
  • 查找第一个子节点【返回节点对象】:
  •       element.firstChild
  • 查找最后一个子节点【返回节点对象】:
  •       element.lastChild
  • 查找指定元素节点的父节点:
  •             element.parentNode
  • 查找指定元素节点的兄弟节点
    • 查找前一个兄弟节点:
  •       node.previousSibling
    • 查找后一个兄弟节点:
  •       node.nextSibling
属性设置：
  • 读取属性值
    • 元素对象.属性名
  • 修改属性值
    • 元素对象.属性名 = 新的属性值

二、Jquery：

    1、概念：
  • $是jQuery的核心函数，jQuery的核心功能都是通过这个函数实现
  • $的四个作用:
    • $(function(){})
      • 相当于window.onload = function(){}
    • $(“选择器”)
      • 根据这个选择器查找元素节点对象
    • $(“html字符串”)
      • 根据这个html字符串创建元素节点对象
    • $(DOM对象)
      • 将DOM对象包装为jQuery对象
2）转换：
  • DOM对象转jQuery对象
    • 使用jQuery核心函数包装DOM对象
    • 例如：var $btnEle = $(btnEle)
    • 声明一个变量指向jQuery对象，那么这个变量习惯上要以$开头
    • jQuery对象转DOM对象
      • 使用数组下标：$btnEle[0]
      • 使用get(index)方法：$btnEle.get(0)
3）属性值的获得和设置：
        dom中使用的是 value  在此 用的是val（）        
        dom中使用的是 innerHTML 在此 用的是html  （）（包含元素）
        dom中使用的是 innerText 在此 用的是text（）  纯文本
 4）选择器：参见速查表
5）属性设置：

        attr(name|properties|key,value|fn)设置或返回被选元素的属性值。

6）添加元素：
        内部: 
                append(content|fn)
                appendTo(content)
                prepend(content|fn)
                prependTo(content)
        外部：
                after(content|fn)
                before(content|fn)
                insertAfter(content)
                insertBefore(content)
7）删除元素：
        empty（）清空子元素
        remove（）删除该元素
8）绑定事件：
        a、bind(type,[data],fn)  为每个匹配元素的特定事件绑定事件处理函数
$("p").bind("click", function(){  /*当每个段落被点击的时候，弹出其文本。*/
alert( $(this).text() );
});

        b、live(type, [data], fn) jQuery 给所有匹配的元素附加一个事件处理函数，即使这个元素是以后再添加进来的也有效。
$('.clickme').live('click', function() {  /*当每个 .clickme 被点击的时候，弹出其文本。即使它是动态新添加的元素*/
    alert("Live handler called.");
});
